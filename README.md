## 🔍 What is Shadow Catcher?

Shadow Catcher is a cross-platform download manager that:

- 📥 Downloads videos, audio, and files from the web
- 🧠 Scans every file with an on-device AI model before saving
- 🚫 Blocks malware, trojans, and malicious streams in real-time
- 🎬 Cleans video streams by filtering malicious packets
- ⚡ Manages bandwidth to protect your system resources
- 🔄 Resumes interrupted downloads automatically
